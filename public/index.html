The tool I needed before I moved to Apple. Sharing clipboard between phone and the PC is very useful to me. This tool generates a unique URL and a QR code to that URL so it can be read from phone. Everytime you save the note, simply refresh the page on the phone to acess the updated note.

The Color Clock

The numbers have taken over our lives from a long time. What if threw that away and used colors instead? The Color Clock aims at exactly that, by displaying time in hex colors. Doesn't it feel great to say Aubergine O'clock instead of 9 O'clock?
